# Drachenlord Fact Pool

This file turns the raw dossier in `research/drachenlord-dossier.de.md` into a quiz-writing working sheet.

## Scope

- This is **derived research**, not a final seed import.
- Facts here are normalized for later question writing.
- Entries should stay conservative and should prefer facts marked as safe or well-supported in the dossier.
- Nothing in this file should be treated as independently re-verified unless that is stated explicitly.

## How to use this file

For each future quiz question:
1. start from one fact entry here
2. re-check the source anchor before turning it into runtime seed data
3. keep the wording simpler than the research note
4. avoid private details and scene-only claims unless clearly marked and justified

## Field guide

- **Fact ID** — stable internal reference
- **Statement** — concise fact claim for question writing
- **Canonical answer** — the intended correct answer
- **Suggested question angles** — possible ways to ask about the fact
- **Distractor angles** — plausible but wrong directions for answer options
- **Dossier anchors** — references back into the dossier structure
- **Reliability in dossier** — how the dossier frames the fact today
- **Quiz status** — whether the dossier marks it as safe, cautionary, or unsuitable
- **Suggested difficulty** — a starting point, not a hard rule
- **Suggested category** — category proposal for future quiz content
- **Suggested tags** — tag ideas for filtering and balancing
